# Tina Nova Defense | Tina新星防御

A high-definition, modern tower defense game built with React, Vite, and Tailwind CSS.

## Features
- **Modern UI**: Sleek, high-definition interface with glassmorphism and neon accents.
- **Level System**: 100 levels of increasing difficulty.
- **Resource Management**: Strategic ammo distribution and battery health.
- **Bilingual**: Supports English and Chinese.
- **Responsive**: Optimized for both desktop and mobile devices.

## Tech Stack
- **Frontend**: React 19, Vite, Tailwind CSS 4
- **Animations**: Motion (formerly Framer Motion)
- **Icons**: Lucide React
- **Audio**: Web Audio API (Synthesized effects)

## Deployment to Vercel

1. **Push to GitHub**:
   - Create a new repository on GitHub.
   - Initialize git in your project: `git init`
   - Add files: `git add .`
   - Commit: `git commit -m "Initial commit"`
   - Link to GitHub: `git remote add origin <your-repo-url>`
   - Push: `git push -u origin main`

2. **Import to Vercel**:
   - Go to [Vercel](https://vercel.com).
   - Click "Add New" -> "Project".
   - Import your GitHub repository.
   - Vercel will automatically detect the Vite configuration.
   - Click "Deploy".

## Local Development

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```
